# Appendix B — Pedagogical Pattern Evidence Mapping

This appendix maps **expressive pathways** to **types of assessment evidence** and **documentation language**. Use with [Classroom Signals Guide](classroom-signals.md) and [Appendix E — E2](appendix-e.md#e2--what-counts-as-evidence-of-understanding).

---

## Pathway → evidence matrix

| Expressive pathway | What may count as evidence | Observable indicators (examples) | Cross-modal criterion |
|--------------------|----------------------------|----------------------------------|------------------------|
| **Written** | Explanation, analysis, argument | Accurate claims, logical structure, use of content vocabulary | Same learning goal as all other rows |
| **Verbal / oral** | Spoken explanation, conference, recording | Coherent account, accurate details, responds to prompts | Accuracy + transfer + clarity for audience |
| **Performative / embodied** | Demonstration, simulation, museum role, lab performance | Actions align to concept; can explain choices when asked | Demonstrates concept, not merely compliance with format |
| **Visual / representational** | Model, diagram, map, composition | Representation matches concept; labels/legend accurate | Same conceptual target as written equivalent |
| **Reflective (delayed)** | Journal after unit, revision reflection, "what changed" | References specific work; shows change in thinking | Sense-making over time, not generic completion |
| **In-role / narrative** | Character portrayal, debate in persona, structured role-play | In-character content accurate; maintains context appropriately | Content knowledge visible *through* narrative frame |

---

## False-negative patterns (when visibility fails)

| Pattern | What the system sees | What may be missed | Structural response |
|---------|----------------------|--------------------|---------------------|
| **Output fixation** | Polished written product | Oral or performative mastery | Add pathway with shared criteria |
| **Speed bias** | Fast responders rewarded | Delayed integrators | Allow asynchronous / reflective evidence |
| **Channel mismatch** | Seat-work behavior | Event-based demonstration | Separate behavior observation from content evidence |
| **AI suspicion** | Uniform suspicion of all text | Whether thinking preceded tool use | Document process; sequence AI after engagement |

## Documentation flow

```
Learning goal (one sentence)
    → Acceptable pathways (2+ only if equivalent)
        → Shared criteria (accuracy, transfer, communication)
            → Standardized language (Classroom Signals / §7.3)
                → Calibration (exemplars, team alignment)
```
