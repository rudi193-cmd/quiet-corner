# Appendix E — Educator Implementation Companion

Universal tools · classroom teacher voice · no product names

**Flow:** E1 (belief) → E2 (design) → E3 (reflection filter) → E4 (principal alignment)

---

## E1 — Seeing Learning: Expanding Assessment Without Lowering Standards

### What this is

A **belief-shift tool** for teachers (PLC, grade team, self-prep before E4).  
Job: answer *"Are we lowering the bar?"* before anyone asks the principal.

**Time:** 10–15 min in a team huddle, or 5 min alone before a parent conference.

### The one idea

> **Narrow evidence produces false negatives.**  
> Widening *what counts as evidence* — while holding the *same learning goal* — makes assessment **more accurate**, not easier.

### Quick contrast (say or post on chart paper)

| Myth | Reframe |
|------|---------|
| "If it's not written, it doesn't count." | Writing is one pathway; it's not the only proof of understanding. |
| "Performance = showing off." | Performance can be **demonstration of concept**, same as an essay. |
| "More ways to show = less rigor." | More ways to show = **fewer kids misread** as not knowing. |
| "AI means we can't trust any work." | AI exposes that we were already **grading outputs**, not always **learning**. |

### Team prompts (pick 2–3)

1. *Where in our building do we already accept multiple pathways?* (music, PE, labs, oral reading, presentations)
2. *Where do we still treat one format as the "real" score?*
3. *Name one student who "gets it" in discussion but underperforms on the written check — what did we miss?*
4. *If we held the standard constant but added one pathway, which unit would be lowest-risk to try?*

### Language to adopt (staff room)

- "**Same goal, different evidence.**"
- "**Under-seen ≠ under-capable.**"
- "We're improving **visibility**, not reducing **expectations**."

### Self-check (before talking to a principal)

| ✓ | You can explain… |
|---|------------------|
| ☐ | The problem is **blind spots in evidence**, not lazy students |
| ☐ | Expanded pathways **raise accuracy** by reducing false negatives |
| ☐ | **Teacher judgment** still interprets whether evidence meets the goal |

**Next:** E2 → E4 when ready.

## E2 — What Counts as Evidence of Understanding?

### What this is

A **practical rubric tool** — turns "I know they know it" into **documented, comparable evidence**.  
Use when designing a unit, revising a rubric, or defending a grade in a conference.

**Time:** 20–30 min to fill once; reuse every unit.

### Step 1 — Name the learning goal (one sentence)

> *Students will `[verb]` `[concept/skill]` by `[observable end state]`.*

**Example:**  
> Students will **explain** how **cause and effect** shapes a historical event **using accurate details and logical connections**.

### Step 2 — List acceptable evidence pathways

Check all that **clearly demonstrate the same goal**:

| Pathway | When it fits | Your context |
|---------|--------------|--------------|
| **Written** | Explanation, analysis, argument | `[essay, paragraph, constructed response]` |
| **Verbal** | Oral explanation, conference, recording | `[explain-out-loud, video, discussion]` |
| **Performative / embodied** | Knowledge shown through action | `[museum role, demo, simulation, lab]` |
| **Visual / representational** | Model, diagram, map, composition | `[poster, graphic organizer, musical theme]` |
| **Reflective (delayed)** | Understanding after processing time | `[journal after revision, "what changed"]` |
| **In-role / narrative** | Content knowledge in character | `[debate in persona, historical monologue]` |

**Rule:** If two pathways don't prove the **same goal**, they aren't equivalent.

### Step 3 — Shared criteria (cross-modal equivalence)

| Criterion | Meets | Not yet |
|-----------|-------|---------|
| **Accuracy** | Core facts/concepts correct | Errors that change meaning |
| **Transfer** | Connects to bigger idea or new example | Isolated facts |
| **Communication** | Clear to audience for this pathway | Missing key parts |

### Step 4 — One-line documentation

Pick from [Classroom Signals Guide](classroom-signals.md):

- *Accurately represented the core concept through non-written expression.*
- *Demonstrated conceptual transfer by connecting ideas.*
- *Articulated understanding after reflection, indicating durable sense-making.*

### Red flags (don't call this evidence)

- Activity completed but **concept not demonstrated**
- Polished output with **no observable thinking**
- **Effort or compliance** substituted for understanding
- Pathway so loose it **doesn't map to the learning goal**

### Self-check

| ✓ | Your rubric… |
|---|----------------|
| ☐ | States **one learning goal** all pathways share |
| ☐ | Lists **2+ pathways** only if each can prove that goal |
| ☐ | Uses **observable criteria**, not student traits |
| ☐ | Has **phrasing** you could show a principal or parent |

## E3 — Reflection as Assessment, Not Homework

### What this is

A **design filter** for reflection tasks — separates reflection from **busywork** and **AI-generated filler**.

**Time:** 10 min to vet one assignment; 15 min to redesign a weak one.

### The distinction

| Reflection as homework | Reflection as assessment |
|------------------------|--------------------------|
| Fills time; checks compliance | Surfaces **sense-making** |
| Same prompt every day | Tied to **specific learning goal** |
| Graded on length | Graded on **visible thinking** |
| Can be done without engaging content | **Requires** engagement first |
| Often first thing students ask AI to write | AI **after** student has thought |

### The sequence (non-negotiable)

```
Engage → Make meaning → Articulate → (optional) AI as secondary support
```

### Prompt vetting — 4 questions

1. Could a student answer **without** doing the learning?
2. Does it reference **specific content** from this unit?
3. Does it ask for **change over time**?
4. Would a **completion grade** be honest?

### Prompt templates

**After engagement:**  
> "After `[activity]`, what is one claim you can defend about `[concept]`? What evidence from our work supports it?"

**Change over time:**  
> "At the start of the unit I thought `[X]`. Now I think `[Y]` because `[reason]`."

**Transfer:**  
> "Where else does `[concept]` show up—in this subject, another subject, or outside school?"

### When AI is in the room

**Appropriate:** clarify wording of *their* draft; rephrase an explanation they hold; practice prompts for self-check.

**Not reflection evidence:** AI writes from a generic prompt; reflection due **before** the learning activity.

### Minimal rubric

| Meets | Not yet |
|-------|---------|
| Names **specific** content from the unit | Generic, any-class wording |
| Shows **reasoning** or change | Restates instructions only |
| Connects to **learning goal** | Off-topic or purely procedural |

## E4 — How to Explain This to Your Principal

### What this is

A **conversation tool** for any classroom teacher — not a speech for one school.

Ask for **recognize learning more accurately** when AI is in the building — without lowering standards, banning tools, or pitching a product.

**Time:** ~4–5 minutes · **Format:** beats in your own words; swap in **your** example in Beat 2.

### Before you go in

**You are asking for:** accuracy, equity, governance, **one pilot step**.

**You are not asking for:** ChatGPT permission, lower expectations, a district purchase.

### Beat sheet

#### Beat 0 — Frame *(~15 sec)*

> "Can I have a few minutes about **how we assess learning** now that AI tools are in students' lives? Not one app. Not lowering the bar."

*Optional:*  
> "I'd like us aligned on **how the school recognizes learning** — something that holds through staffing or leadership changes."

#### Beat 1 — Problem *(~20 sec)*

> "A lot of our assessment favors **fast, visible output**. That misses learning that shows up other ways. AI didn't create that blind spot; it **made it harder to ignore**."

#### Beat 2 — Your proof *(~60 sec)*

**Template:**

> "Recently, during `[event or unit]`, I saw a student demonstrate `[learning goal]` through `[how it showed up]`. Our usual format would have **under-captured** that—not because the standard was wrong, but because the **evidence was narrow**."

**Pathway prompts:** performative/embodied · verbal · reflective (delayed) · in-role/narrative.

*No student names. No deficit labels. Describe **what you observed**.*

#### Beat 3 — Standards stay; evidence widens *(~45 sec)*

> "Same learning goals, **more accurate** evidence—performance, spoken explanation, reflection, in-character demonstration when it shows the concept. AI: **secondary support after engagement**—not primary thinking, not auto-grading. **My judgment stays central.** Transparency is **governance**, not surveillance."

#### Beat 4 — Shared language *(~30 sec)*

> "Shared phrases for report cards and conferences—observation, not labels."  
> *(Offer [Classroom Signals](classroom-signals.md) phrases as leave-behind.)*

#### Beat 5 — What you're not asking *(~20 sec)*

> "Not banning AI, removing deadlines, unfunded workload, buying a platform, or replacing judgment with a detector."

#### Beat 6 — One clear ask *(~30 sec)*

Pick **one:**

| Ask | Script |
|-----|--------|
| **Pilot** | "Pilot **two expressive pathways** for the same goal in `[grade/unit]`—written **or** spoken/performative—with **shared criteria**?" |
| **Policy language** | "Frame AI guidance around **visibility and governance**, not only output detection?" |
| **Calibration** | "One team session on assessment language and exemplars before we pick tools?" |

#### Beat 7 — Close *(~15 sec)*

> "I can share anonymized work or draft rubric language with `[team]`. I'm asking for **accuracy and fairness**—for students already showing understanding our formats don't capture."

### After the meeting — self-check

| ✓ | Principal should believe… |
|---|---------------------------|
| ☐ | Problem is **assessment design**, not "kids + AI" |
| ☐ | You want **accuracy**, not easiness |
| ☐ | **Teacher judgment** stays central |
| ☐ | This is **equity** — under-seen ≠ under-capable |
| ☐ | Transparency = **governance**, not surveillance |
| ☐ | **One concrete next step** agreed |

### Optional follow-up email

**Subject:** Assessment visibility — follow-up

> Thanks for the time today. I'm concerned our formats capture **visible output** more reliably than **understanding**, and AI intensifies that mismatch. I'm requesting a **small pilot** (or **shared assessment language**) with the same standards and shared criteria. Attached: sample report-card phrases. Happy to meet with `[team]` on rubric language.
